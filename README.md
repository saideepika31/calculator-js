# calculator-js
A simple js based calculator which performs the basic calculations . On the given input !
Built using HTML, CSS, and JavaScript.
This project supports basic arithmetic operations, floating-point numbers, percentage calculations, and clear/backspace functionality.


FEATURES:
Addition
Subtraction
Multiplication
Division
Supports decimal values
Backspace function
Clear function
Percentage calculation
Basic error handling (invalid expressions handled gracefully)
Responsive layout (works on mobile & desktop)


Tech Stack:
**HTML**
**CSS**
**JavaScript (ES6)**
